## Requirements

### Dependencies
- **SDL2** development libraries
- **C++11** compatible compiler (GCC, Clang, MSVC)
- **Make** build system

### Installing Dependencies

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install build-essential libsdl2-dev libsdl2-ttf-dev libsdl2-image-dev libsdl2-mixer-dev
```

## Building

### Quick Build
```bash
make
```
